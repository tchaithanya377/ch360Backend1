Usage Examples
Get All Courses
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "{{ api_base_url }}courses/"
Create a New Course
curl -X POST \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"code":"CS101","title":"Intro to CS","credits":3}' \
  "{{ api_base_url }}courses/"
Get Courses with Filtering
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "{{ api_base_url }}courses/?status=ACTIVE&level=UNDERGRADUATE"
Search Courses
curl -H "Authorization: Bearer YOUR_TOKEN" \
  "{{ api_base_url }}courses/?search=computer"
